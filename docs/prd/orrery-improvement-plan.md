## 📋 프로젝트 개요

### 현재 상태

**Orrery**는 사주팔자 + 자미두수 + 서양 점성술을 브라우저에서 계산하는 클라이언트 사이드 명리학 도구예요.

| 항목 | 현황 |
|------|------|
| 코어 엔진 | 10,200줄 TypeScript (성숙) |
| 웹앱 | 2,600줄 React (기본 기능 완성) |
| 테스트 | 222줄 (핵심 계산 검증) |
| npm 패키지 | @orrery/core v0.3.0 |

### 차별화 포인트

전 세계에서 **유일하게** 3대 명리 체계를 하나의 TypeScript 엔진으로 통합 계산해요:
- 동양: 사주팔자(한국/중국), 자미두수(중국)
- 서양: Natal Chart (Swiss Ephemeris 이론)

### 해결해야 할 문제

1. **사용성 부족**: 계산만 하고 끝. 해석, 저장, 비교 기능 없음
2. **접근성 제한**: 한국어 전용, 라이트 모드만, 인쇄/내보내기 불가
3. **수익 모델 부재**: 무료 서비스, 지속 가능성 낮음
4. **경쟁 서비스 대비**: Co-Star, 점신 대비 UX 열세

## 🚀 배포: Vercel 이전

### 왜 Vercel인가?

| 항목 | GitHub Pages | Vercel |
|------|-------------|--------|
| 빌드 속도 | 느림 (Actions) | 빠름 (자체 빌드) |
| CDN | GitHub CDN | Edge Network (더 빠름) |
| Analytics | 없음 | 내장 (무료) |
| 프리뷰 배포 | 없음 | PR마다 자동 |
| 가격 | 무료 | 무료 (Hobby) |

### 배포 설정

```json
// vercel.json
{
  "buildCommand": "bun run build",
  "outputDirectory": "dist",
  "installCommand": "bun install",
  "framework": "vite"
}
```

### 도메인 설정

1. **sky.told.me 유지**: Vercel에서 커스텀 도메인 연결
2. **대체 도메인**: orrery.vercel.app (자동 생성)

  slot: string
  format?: 'auto' | 'horizontal' | 'vertical'
  className?: string
}

export function AdBanner({ slot, format = 'auto', className }: AdBannerProps) {
  useEffect(() => {
    try {
      (window.adsbygoogle = window.adsbygoogle || []).push({})
    } catch {}

}
