# 명운판 수익화 전략

## 📊 현재 상태 분석

### 강점
- ✅ AdSense 정책 준수 완료 (콘텐츠 97/100 품질)
- ✅ JavaScript 에러 0건 (안정적 광고 로딩 보장)
- ✅ 모바일 반응형 디자인
- ✅ 클라이언트 사이드 계산 (빠른 페이지 로드)
- ✅ 유기적 트래픽 잠재력 (명리학 관련 검색어)

### 약점
- ⚠️ 광고 슬롯이 1개만 활성화
- ⚠️ 쿠팡 파트너스 미연동
- ⚠️ 영어 서비스 미활성화 (글로벌 트래픽 손실)
- ⚠️ SEO 최적화 부족
- ⚠️ 사용자 참여 유도 장치 부족

## 🎯 수익화 3단계 로드맵

### Phase 1: 광고 최적화 (즉시 시작 가능)

#### 1-1. AdSense 광고 슬롯 확장

**추가해야 할 광고 위치:**

| 위치 | 슬롯 ID | 형식 | 예상 CTR | 우선순위 |
|------|---------|------|----------|----------|
| Hero 섹션 하단 | `SLOT_HERO` | Display (horizontal) | 1.5-2.0% | P0 |
| 입력 폼 하단 | `SLOT_FORM` | Display (horizontal) | 2.0-3.0% | P0 |
| 결과 상단 (탭 위) | `SLOT_RESULT_TOP` | Display (horizontal) | 3.0-4.0% | P0 |
| 각 탭 결과 하단 | `SLOT_SAJU_END` | Display (rectangle) | 2.5-3.5% | P1 |
| 가이드 상단 | `SLOT_GUIDE` | Display (horizontal) | 1.0-1.5% | P1 |
| 사이드바 (데스크톱) | `SLOT_SIDEBAR` | Multiplex | 2.0-3.0% | P2 |
| 앵커 광고 (모바일) | `SLOT_ANCHOR` | Anchor | 4.0-6.0% | P1 |

**예상 수익 증가:**
- 현재 1개 슬롯 → 5-7개 슬롯 확장
- **예상 RPM (페이지당 수익) 증가: 300-500%**

#### 1-2. 쿠팡 파트너스 통합

**상품 추천 전략:**

```markdown
### 배치 위치
1. **결과 페이지 하단** (P0)
   - "명리학 공부에 도움되는 추천 도서"
   - 타겟: 심화 학습 관심 사용자
   - 예상 전환율: 0.5-1.0%

2. **가이드 섹션** (P1)
   - "사주팔자 해석 가이드북"
   - "자미두수 입문서"
   - 예상 전환율: 1.0-2.0%

3. **AI 해석 모달** (P2)
   - "AI 시대의 명리학"
   - 타겟: 기술 + 명리학 관심층
   - 예상 전환율: 0.3-0.5%

### 추천 상품 카테고리
- 명리학 서적 (사주팔자, 자미두수, 점성술)
- 만세력/명리 소프트웨어
- 타로 카드/오라클 카드
- 명상/영성 관련 도서
- 운세 관련 문구류 (다이어리, 플래너)
```

**구현 방식:**
```typescript
// src/components/CoupangPartner.tsx
interface CoupangProductProps {
  productId: string
  title: string
  imageUrl: string
  price: number
  category: 'book' | 'software' | 'stationery'
}

// 쿠팡 파트너스 링크 생성 함수
function generateCoupangLink(productId: string): string {
  return `https://coupa.ng/YOUR_PARTNER_ID/${productId}`
}
```

#### 1-3. 광고 성과 측정 시스템

**Google Analytics 4 연동:**
```typescript
// 광고 클릭 이벤트 추적
gtag('event', 'ad_click', {
  'ad_slot': 'SLOT_RESULT_TOP',
  'ad_format': 'display',
  'user_language': 'ko'
})

// 쿠팡 상품 클릭 추적
gtag('event', 'affiliate_click', {
  'product_category': 'book',
  'product_id': 'ABC123'
})
```

**성과 지표 대시보드:**
- 슬롯별 CTR (Click-Through Rate)
- 슬롯별 RPM (Revenue Per Mille)
- 쿠팡 전환율
- 사용자 세그먼트별 광고 성과

#### 2-2. 사용자 참여 유도

**신규 기능 제안:**

1. **일간 운세 (Daily Fortune)**
   - 매일 방문 유도 → 광고 노출 증가
   - 푸시 알림 연동 (PWA)

2. **커뮤니티 기능**
   - 사용자 리뷰/후기
   - 질문/답변 게시판
   - UGC (User Generated Content) → SEO 향상

3. **소셜 공유 인센티브**
   - "친구에게 공유하고 AI 해석 무료로 받기"
   - 바이럴 마케팅 효과

4. **이메일 수집**
   - "주간 운세 뉴스레터 구독"
   - 재방문율 향상

#### 2-3. 백링크 구축

**전략:**
- 명리학 커뮤니티 (네이버 카페, 인스타툰) 홍보
- 인플루언서 협업 (역술인 유튜버)
- 프레스 릴리스 (오픈소스 명리학 플랫폼)

### Phase 3: 프리미엄 수익화 (3-6개월)

#### 3-1. 프리미엄 플랜

**무료 vs 프리미엄 비교:**

| 기능 | 무료 | 프리미엄 |
|------|------|----------|
| 기본 계산 (사주/자미/출생차트) | ✅ | ✅ |
| AI 해석 (기본) | ✅ 1회/일 | ✅ 무제한 |
| AI 해석 (심화) | ❌ | ✅ |
| PDF 리포트 다운로드 | ❌ | ✅ |
| 광고 제거 | ❌ | ✅ |
| 이메일 상담 | ❌ | ✅ (월 1회) |
| 대운/세운 예측 | ❌ | ✅ |
| 궁합 분석 (상세) | ❌ | ✅ |

**가격 전략:**
- 월 구독: ₩9,900
- 연 구독: ₩99,000 (17% 할인)
- 평생 이용권: ₩299,000

**예상 전환율:**
- 무료 사용자 → 프리미엄: 0.5-1.0%
- 월 방문자 10,000명 기준 → 50-100명 구독
- 예상 월 수익: ₩495,000 - ₩990,000

**모델:**
